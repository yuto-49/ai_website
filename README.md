## Getting Started

### Quick Start

1. **Install dependencies:**
   ```bash
   # Backend
   pip install -r requirements.txt

   # Frontend
   npm install
   ```

2. **Set up API key:**
   ```bash
   # Copy example env file
   cp env.example .env

   # Add your Anthropic API key to .env
   ANTHROPIC_API_KEY=your-key-here
   ```

3. **Start the backend:**
   ```bash
   python server.py
   # Server will run on http://localhost:5001
   ```

4. **Start the frontend:**
   ```bash
   npm run dev
   # Frontend will run on http://localhost:5173
   ```

5. **Start chatting!**
   - Type a message and watch the AI respond
   - Brainstorming agent automatically generates topic ideas
   - Navigate to `/map` to see the visual conversation tree
   - Purple nodes with âœ¨ are AI-generated topic suggestions!

## Agentic System

### What is it?

The application includes an **extensible agentic system** that allows AI agents to work autonomously in the background, enhancing conversations with additional capabilities.

### Current Agents

#### Brainstorming Agent
- **Purpose**: Generates 1-3 divergent, interesting topic ideas from your conversations
- **Execution**: Runs in parallel with your main chat
- **Visual**: Creates purple nodes with sparkle badges (âœ¨) in the NodeMap
- **Usage**: Automatic - runs on every message

### Adding Your Own Agents

The system is designed to be easily extensible. See `AGENTIC_SYSTEM.md` for detailed documentation, or `AGENT_QUICK_START.md` for a 5-minute guide to creating your first agent.

**Quick example:**
```python
# In server.py
"my_agent": {
    "name": "My Agent",
    "model": "claude-3-haiku-20240307",
    "enabled": True,
    "execution": "parallel",
    "description": "What your agent does"
}
```

### Agent Ideas

- **Summarizer**: Automatically summarize long conversations
- **Fact Checker**: Verify factual claims in real-time
- **Code Reviewer**: Review code snippets and suggest improvements
- **Question Generator**: Generate thoughtful follow-up questions
- **Research Assistant**: Fetch additional context (with tool integration)

## Customization

### Colors
Edit the CSS variables in `src/App.css` to customize the color scheme:

```css
:root {
    --accent-primary: #667eea;
    --accent-secondary: #764ba2;
    /* ... more variables */
}
```

### Agent Configuration
Edit agent behavior in `server.py`:

```python
"brainstorming": {
    "temperature": 0.9,  # Creativity level
    "max_tokens": 300,   # Response length
    "enabled": True      # Enable/disable
}
```
